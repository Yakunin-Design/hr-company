# Starting scripts

## **For linux:**
### Starting typescript compiler
```bash
    npm run ts 
```

### Starting the nodemon server
```bash
    npm run dev
```

## **For windows:**
### Starting typescript compiler
```bash
    npm run ts-w 
```

### Starting the nodemon server
```bash
    npm run dev-w
```

## Run project (if it is compiled)
```bash
    npm start
```
